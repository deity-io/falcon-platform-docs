Deployment is handled through **Falcon Cloud**, all you need to do is push your code.

Read more about [Falcon Cloud](/console/cloud/about)

To check your build code locally you can run the `build` & `start` commands with the `:prod` flag from both the `server` and `client`.

import { TabsContainer, Tabs, TabItem } from '@site/src/components/Tabs';
import CodeBlock from '@theme/CodeBlock';

<TabsContainer>
  <Tabs
    variant="secondary"
    defaultValue="npm"
    values={[
      { label: 'npm', value: 'npm' },
      { label: 'Yarn', value: 'yarn' }
    ]}
  >
    <TabItem value="npm">
      <CodeBlock
        language="bash"
        children={`
npm build:prod
npm start:prod
      `.trim()}
      />
    </TabItem>
    <TabItem value="yarn">
      <CodeBlock
        language="bash"
        children={`
yarn build:prod
yarn start:prod
      `.trim()}
      />
    </TabItem>
  </Tabs>
</TabsContainer>
;

Find out more about our [CLI commands](/platform/client/getting-started#build)
