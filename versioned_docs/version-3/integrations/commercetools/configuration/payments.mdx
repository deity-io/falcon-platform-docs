## Overview

With our existing setup all payments should go through our [payment service](/docs/payments).

In order to correctly store payment data against your payments in Commercetools, we must add a custom type to Commercetools.

## Configuration

Installation custom types in Commercetools requires us to run an API request.

We recommend using [Postman](https://www.postman.com/) to do this.
To set up Postman for Commercetools, you can make use of the Commercetools collection.
We explained the [installation process here](/docs/integrations/commercetools/getting-started#commercetools-postman-collection).

Once the collection is configured, the final step is to run the request `Project -> Types -> Create Type` (`{{host}}/{{project-key}}/types`).
