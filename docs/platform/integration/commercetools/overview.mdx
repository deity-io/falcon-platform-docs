## Introduction
Commercetools is a headless e-commerce platform built on MACH principles (microservice-based, API-first, cloud-native and headless).

- [Commercetools Website](https://commercetools.com/)

## Integration Concepts

- ctpSession - Hold http session data
- ctpClient - Connect to SDK in name of user or integration, using ctpSession for authentication

## Default datasources
Out of the box, our Commercetools integration
- CategoryDatasource
- CatalogDatasource
- CustomerDatasource
- AddressDatasource
- OrderDatasource
- CartDatasource
- PaymentDatasource
