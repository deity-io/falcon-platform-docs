## Overview

All requests the go through our cloud hosting a proxied by `nginx`. 

We append headers based on the GeoIP to these requests.

```
X-GeoIP-Country
X-GeoIP-Country-Name
X-GeoIP-Region-Name
X-GeoIP-City
X-GeoIP-Postal-Code
```
