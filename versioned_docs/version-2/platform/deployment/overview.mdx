Deployment is handled through **Falcon Cloud**, all you need to do is push your code.

Read more about [Falcon Cloud](/docs/platform/cloud/about)

To check your build code locally you can run the `build` & `start` commands with the `:prod` flag from both the `server` and `client`.

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs
  defaultValue="npm"
  values={[
    { label: 'NPM', value: 'npm', },
    { label: 'Yarn', value: 'yarn', }
  ]
}>
<TabItem value="npm">

```bash
npm build:prod
npm start:prod
```

</TabItem>
<TabItem value="yarn">

```bash
yarn build:prod
yarn start:prod
```

</TabItem>
</Tabs>

Find out more about our [CLI commands](/docs/platform/client/getting-started#build)
